# 010: 2Dキーポイント信頼度分析

## 目的

三角測量の前提となる2Dキーポイント情報（各カメラのpose推定結果）の信頼度を精査し、3D精度に悪影響を与えるパターンを特定する。

## 背景

- 現在の三角測量は `likelihood_threshold = 0.4` でフィルタしているが、閾値を超えた低信頼度データが混入している可能性がある
- cam01/cam03の除外率35-39%の原因が、キャリブレーション精度だけでなく2Dポーズ推定の品質にも起因する可能性
- 004（likelihood_threshold最適化）の事前調査としても有用

## 対象データ

`Pose2Sim/20260227-dgtw2-lab2/pose/cam{01-04}_json/*.json`

- 4カメラ × 全フレーム（cam01: 1800, cam02: 1794, cam03: 1799, cam04: 1794）
- 26キーポイント（HALPE_26）
- JSON形式: `pose_keypoints_2d` = [x, y, confidence] × 26
- 各フレーム1人のみ（`person_id = [-1]`）
- 信頼度の値域: 0.397〜1.103（1.0超あり、0.0は無し）

## カメラ配置分析（2026-03-02）

### カメラ位置と向き

| カメラ | X (m) | Y (m) | Z (m) | 焦点距離 (fx) | レンズ種別 |
|--------|--------|--------|--------|---------------|-----------|
| cam01 | 3.879 | 0.538 | 1.052 | 2014 | 望遠 |
| cam02 | 2.441 | -0.206 | 1.026 | 1189 | 広角 |
| cam03 | 3.280 | 1.878 | 1.072 | 2037 | 望遠 |
| cam04 | 1.558 | 2.553 | 0.844 | 1182 | 広角 |

全カメラの光軸が-X方向（原点方向）を向いており、被験者は原点付近を歩行。

### 配置の特徴

```
         Y
         ^
  cam04  |  cam03        cam01
  (広角)  |  (望遠)       (望遠)
         |
         |       cam02
         |       (広角)
 --------+--------------------> X
         |
     [被験者エリア]
```

- **片側半円〜L字型配置**: 4台すべてが+X側に集中し、-X方向（原点=被験者エリア）を向く
- **全カメラが同じ方向**: 被験者を囲む配置ではなく、一方向からの撮影
- **2ペア構成**: cam01+cam02（下側）、cam03+cam04（上側）で各ペア内に望遠+広角1台ずつ
- **高さはほぼ均一**: Z=0.84〜1.07m（約1m）

### 低信頼度の原因: カメラ配置による死角

片側配置のため、**被験者の左半身がカメラ群から見て体の奥側（遮蔽側）になりやすい**。これが左側キーポイントの系統的な低信頼度を引き起こしている。

| 問題 | 原因 |
|------|------|
| 左側キーポイント低信頼度（LWrist/LElbow/LHip） | 全カメラが片側配置→左半身が体幹による自己遮蔽を受ける |
| cam01/cam03の高除外率(35-39%) | 望遠レンズ+片側配置で、遮蔽時の2D推定精度が特に悪い |
| 閾値引き上げの効果が限定的 | 問題は2D推定自体の構造的限界であり、閾値変更では対処困難 |

### 精度改善への影響

**左側キーポイントの精度問題はカメラ配置由来のデータ品質限界**であり、以下の対策では根本的に改善しない:

- 004: likelihood_threshold最適化 → 構造的な低信頼度データは閾値変更で救えない
- 005: min_cameras引き上げ → 左側キーポイントは全カメラで低信頼度のため、台数制約を強化しても改善しない
- 006: reproj_error_threshold最適化 → 2D推定の精度自体が低いため、再投影誤差フィルタでは限定的

**有効な可能性がある対策**:
- フィルタリング（008）: 三角測量後の3Dデータに対するフィルタリング強化で、低品質区間を平滑化
- markerAugmentation: LSTMによるマーカー拡張で、低品質キーポイントを他のキーポイントから補完
- 今後のデータ収集: カメラを被験者の周囲に分散配置（対面配置の追加）
